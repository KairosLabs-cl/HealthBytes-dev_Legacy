# System Overview

HealthBytes uses a mobile frontend, backend API, relational database, cache, and external integrations.

Current stack:

- Frontend: React Native, Expo, TypeScript, NativeWind, Zustand, React Query.
- Backend: FastAPI, Python, SQLAlchemy async, PostgreSQL, Redis.
- Auth: Clerk plus JWT-related backend flows.
- Payments: Mercado Pago.
- Email: Resend.
- Observability: Sentry.

This file should stay high-level. Backend, frontend, infrastructure, and data details live in sibling docs.
