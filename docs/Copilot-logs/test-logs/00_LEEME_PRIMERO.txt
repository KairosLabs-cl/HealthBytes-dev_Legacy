COMANDOS QUE NECESITAS (copiapega):

Ejecutar todos los tests:
$ cd Backend
$ pytest tests/ -v

Ver qué test falla exactamente:
$ pytest tests/test_api/test_<archivo>.py -v -s

Debuggear un test que falla:
$ pytest tests/test_api/test_<archivo>.py::test_<nombre> -v --tb=short

Ver cobertura de código:
$ pytest tests/ --cov=app --cov-html=htmlcov
$ # Luego abrir: htmlcov/index.html
