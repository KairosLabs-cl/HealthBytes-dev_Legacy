# Product Documentation

Product truth for HealthBytes.

Read this section to understand what HealthBytes is building, who it serves, and how product decisions should be framed.

Primary product frame:

- Pilot users: people managing diabetes, dyslipidemia, and arterial hypertension.
- Initial MVP value: product discovery plus trust.
- Commercial model: mixed direct sale and external provider redirection.
- Recommendation model: HealthBytes recommends based on available data; final decision remains with the user.
