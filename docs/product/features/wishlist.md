# ❤️ Wishlist (Lista de Deseos)

Permite a los usuarios guardar productos para comprarlos después.

## Características
- **Alternar Favorito:** Botón de corazón en tarjetas de producto y detalle.
- **Persistencia:** Guardado local o sincronizado con backend (según autenticación).
- **Vista Dedicada:** Pantalla de "Mis Favoritos".
