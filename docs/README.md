# HealthBytes Documentation

This folder is the documentation entrypoint for HealthBytes.

## Main Sections

| Need | Go to |
| --- | --- |
| Product vision and users | `product/` |
| MVP and roadmap | `roadmap/` |
| Architecture and setup | `development/` |
| AI agents, prompts, inspections, skills | `ai/` |
| Weekly executive reporting | `executive/` |
| QA, security, release process | `operations/` |
| Historical docs during migration | `legacy/` |

## Rules

- Keep root docs short.
- Update the single source of truth before updating indexes.
- Put AI-generated inspections in `ai/inspections/`.
- Move old docs to `legacy/` before rewriting them.
- Do not duplicate roadmap/status across multiple files.
