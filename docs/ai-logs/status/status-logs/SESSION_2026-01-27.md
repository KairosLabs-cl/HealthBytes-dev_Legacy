## 🎯 Problemas Identificados y Resueltos

### 1. Error `import.meta` en Expo Web

**Problema**:
```
Uncaught SyntaxError: import.meta may only appear in a module
```

**Causa**: Las dependencias modernas (Zustand v5) usan sintaxis ESM (`import.meta`) que Metro bundler no procesaba correctamente para web.

**Solución**:
- Actualizado [`metro.config.js`](file:///c:/Users/benja/Proyects/Code/Work/HealthBytes-dev/frontend/metro.config.js)
- Añadida configuración: `resolver.unstable_conditionNames = ['browser', 'require', 'react-native']`
- Fuerza a Metro a priorizar versiones CommonJS sobre ESM

**Resultado**: ✅ Expo Web funciona correctamente

---

### 2. Error de Dark Mode en NativeWind

**Problema**:
```
Uncaught Error: Cannot manually set color scheme, as dark mode is type 'media'.
Please use StyleSheet.setFlag('darkMode', 'class')
```

**Causa**: `tailwind.config.js` tenía `darkMode: 'media'` que genera conflicto con `react-native-css-interop`.

**Solución**:
- Actualizado [`tailwind.config.js`](file:///c:/Users/benja/Proyects/Code/Work/HealthBytes-dev/frontend/tailwind.config.js)
- Cambiado de `darkMode: process.env.DARK_MODE ? process.env.DARK_MODE : 'media'` a `darkMode: 'class'`

**Resultado**: ✅ Sin errores de color scheme

### 4. Dependencias Desactualizadas

**Problemas**:
- Peer dependency warnings de React 19
- `expo-router` requería `@expo/metro-runtime` más reciente
- Faltaba `react-native-svg` (requerido por iconos)
- Faltaba `@testing-library/jest-native` (tests no funcionaban)
- `react-native-worklets` en versión incorrecta

**Soluciones Aplicadas**:

#### Actualizaciones de versión:
```json
// package.json
"expo-router": "~6.0.22",        // antes: ~4.0.22
"expo-linking": "~8.0.11",       // antes: ~7.0.5
"@expo/metro-runtime": "^6.1.2", // añadido
"react-native-svg": "15.15.1",   // añadido
"react-native-worklets": "0.5.1" // antes: 0.7.2 (downgrade)
```

#### Peer Dependencies:
```json
"peerDependencyRules": {
  "allowedVersions": {
    "react": "18 || 19",      // antes: "18"
    "react-dom": "18 || 19"   // antes: "18"
  }
}
```

**Comandos ejecutados**:
```bash
# Fix automático de Expo
npx expo install --fix

# Instalación manual de faltantes
pnpm add react-native-svg
pnpm add -D @testing-library/jest-native

# Downgrade necesario
pnpm add react-native-worklets@0.5.1
```

**Resultado**: ✅ `pnpm install` sin warnings
